# Architecture Design Admin Board

## 🌐 架構設計管理界面

現代化的Web管理界面，提供直觀的架構設計分析功能。

### ✨ 功能特性
- 🌐 響應式Web界面
- 📱 多設備支持
- ⚡ 實時分析結果
- 📊 可視化展示

### 🏗️ 組件結構
- `frontend/` - 前端界面
- `backend/` - 後端API服務
- `config/` - 配置文件
- `docs/` - 界面文檔

### 🚀 啟動方法
```bash
# 啟動後端服務
cd backend
python3.11 ui_backend_server.py

# 訪問界面
http://localhost:5003
```

### 📋 功能模塊
1. 需求輸入界面
2. 分析結果展示
3. 歷史記錄查看
4. 系統狀態監控

### 🔧 配置
- 默認端口: 5003
- 支持CORS跨域
- 自動錯誤處理

### 📊 特性
- 現代化UI設計
- 實時狀態更新
- 移動端適配
- 無障礙支持
